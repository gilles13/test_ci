# Tutoriel

En mode *trash*

1. Si nécessaire, se créer un compte [`github`](https://github.com/)

2. Créer un projet *via* l'IHM (cliquer sur New en haut à gauche), sans README.md

3. Créer les fichiers nécessaires :

  + `_config.yml`
  + `_toc.yml`
  + `conf.py`
  + `requirements.txt`
  + `index.md`
  + autant de *notebooks* que nécessaire
